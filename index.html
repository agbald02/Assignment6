Getting Started with Angular

Before you begin, you should already have knowledge of the following:

  • HTML
  • CSS
  • Javascript
  • JQuery

If you do not yet understand those concepts, there are many examples and other tutorials you can find online. However, if you have an understanding of these concepts, let's continue.


In order to use Angular, you first have to reference it within your html document. You can do so by putting the following code in the HEAD section:

<script src="http://ajax.googleapis.com/ajax/libs/angularjs/1.2.26/angular.min.js"></script>

How Angular Works

Directives

Angular is coupled with HTML by the use of ng-directives.

The ng-app directive defines an AngularJS application.

The ng-model directive binds the value of HTML controls (input, select, textarea) to application data.

The ng-bind directive binds application data to the HTML view.



In order to make a page HTML5 valid, use "data-ng-" instead of "ng-".


Expressions

AngularJS expressions are written inside double braces: {{ expression }}.

AngularJS expressions bind data to HTML the same way as the ng-bind directive.

AngularJS will "output" data exactly where the expression is written.



You can display an expression in certain formats using a filter in the following way:

<p My first expression: {{ 5 + 5 }} | currency></p>

Controllers

AngularJS applications are controlled by controllers.

The ng-controller directive defines the controller.

The controller code will execute when the page loads.
